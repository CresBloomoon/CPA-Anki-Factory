# CPA-Anki-Factory

## 概要
CPA-Anki-Factory は、公認会計士試験（論文式）において  
**「暗記」ではなく「答案を再現できること」** を目的とした Anki デッキを、AI を用いて半自動生成するための設計リポジトリです。

本リポジトリでは、以下を管理します。

- AI（Manus 等）に投入する **プロンプト仕様**
- 出力される Anki カードの **構造・表示・思想**
- 設計判断を記録する **ADR（Architecture Decision Record）**

---

## 基本思想
- Anki は暗記ツールではない
- Anki は **論文答案の「型」を刷り込むための道具** である
- 論証は「覚える」のではなく **「再現できる」状態** をゴールとする

---

## 想定ユースケース
- 論文対策集・テキスト PDF を AI に解析させる
- 各論証を「問い → 答案構造 → 要約」の形で Anki 化
- A/B/C ランクで論点の重要度を制御
- 直前期には A ランク論証のみで高速回転

---

## ディレクトリ構成（予定）
```
CPA-Anki-Factory/
├─ ADR/                # 設計判断（日本語）
├─ prompts/            # Manus投入用プロンプト
├─ examples/           # 生成例・スクリーンショット
└─ README.md
```

---

## 対象科目（想定）
- 財務理論
- 管理会計論
- 監査論
- 企業法
- 租税法

※ 科目ごとにプロンプトを微調整することを想定

---

## 補足
このリポジトリは「AIで楽をする」ことを目的としたものではありません。  
**AI を、学習設計の実装者として使う**ための設計書置き場です。
